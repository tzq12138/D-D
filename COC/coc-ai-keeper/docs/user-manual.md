# COC AI Keeper 用户手册

## 什么是 COC AI Keeper？

COC AI Keeper 是一个本机多人联机网页应用，让 AI 扮演《克苏鲁的呼唤》第七版的守秘人（Keeper），推进调查故事、扮演 NPC、控制恐怖氛围和信息释放。系统负责房间管理、角色状态、资料检索、掷骰与 COC 7 版核心规则判定。

**核心特性**：
- AI 守秘人自动推进故事（支持 DeepSeek API 或模拟模式）
- 完整的 COC 7e 规则引擎（掷骰、检定、SAN 检定）
- 多人实时联机（守秘人 + 多名玩家）
- 角色卡导入（支持 XLSX 格式）
- 资料库检索（AI 自动引用相关资料）

## 快速开始

### 第一步：启动系统

**方式一：一键启动（推荐）**

Windows 用户：
```
双击 start.ps1 文件
```

macOS / Linux 用户：
```bash
chmod +x start.sh
./start.sh
```

**方式二：手动启动**

```bash
cd coc-ai-keeper
npm install
npm run dev
```

### 第二步：打开浏览器

启动后，系统会自动打开浏览器。如果没有自动打开，请手动访问：
- 玩家界面：http://127.0.0.1:5173
- 守秘人界面：http://127.0.0.1:5173/keeper/[房间ID]

### 第三步：创建房间

1. 在首页点击"创建调查房间"
2. 输入房间名称（如"阿卡姆图书馆暗门"）
3. 系统会自动创建房间并生成房间 ID

### 第四步：加入房间

**玩家加入**：
1. 访问房间链接：`http://127.0.0.1:5173/room/[房间ID]`
2. 输入你的名字（如"温特斯"）
3. 选择角色类型（玩家）
4. 点击"加入调查"

**守秘人加入**：
1. 访问守秘人控制台：`http://127.0.0.1:5173/keeper/[房间ID]`
2. 输入守秘人名称（通常为"Keeper"）
3. 点击"进入控制台"

## 玩家操作指南

### 提交行动

1. 在消息输入框中描述你的行动
2. 点击"提交行动"或按 Enter
3. AI 守秘人会根据你的行动生成叙事回应

**行动示例**：
- "我仔细搜查图书馆的书架，寻找关于'暗门'的线索"
- "我询问馆长关于阿尔米蒂奇失踪的情况"
- "我检查工作桌上的笔记"

### 查看角色卡

- 你的角色卡会显示在页面侧边栏
- 角色卡包含：属性值、技能值、物品列表、状态信息
- HP（生命值）和 SAN（理智值）会实时更新

### 掷骰

当 AI 守秘人要求你进行检定时：
1. 系统会在消息中显示检定卡片
2. 点击"掷骰"按钮
3. 系统会自动计算结果并显示成功/失败
4. AI 守秘人会根据结果生成后续叙事

### 查看消息流

- 所有消息会按时间顺序显示在消息流中
- 不同类型的消息有不同的颜色标记：
  - 白色：普通消息
  - 绿色：AI 守秘人叙事
  - 金色：掷骰结果
  - 灰色：系统消息

## 常见问题

### 我没有 DeepSeek API Key 可以使用吗？

可以！系统会自动使用模拟守秘人。模拟守秘人虽然不如真实 AI 智能，但足以演示完整的游戏流程。

### 如何使用 DeepSeek API？

1. 访问 https://platform.deepseek.com/ 注册并获取 API Key
2. 在项目根目录创建 `.env` 文件
3. 添加一行：`DEEPSEEK_API_KEY=sk-your-key-here`
4. 重启系统

### 角色卡如何导入？

1. 准备 COC 7e 格式的 XLSX 角色卡
2. 在守秘人控制台或玩家界面找到"导入角色卡"按钮
3. 选择你的 XLSX 文件
4. 系统会自动解析并导入角色数据

### 资料库如何使用？

1. 将你的资料文件（.md、.txt、.pdf）放入 `sources` 目录
2. 在守秘人控制台点击"扫描资料库"
3. 系统会自动分块并索引资料
4. 当玩家提交行动时，AI 会自动检索相关资料

### 网络断开怎么办？

如果 Socket.IO 连接断开：
- 页面会显示连接错误提示
- 系统会自动尝试重连
- 重连成功后会自动同步最新状态
