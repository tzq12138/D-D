# COC AI Keeper - 开发阶段跟踪表

> **项目目标**：构建一个本机演示版网页应用，AI 作为守秘人（KP）推进故事，软件负责多人房间、角色状态、资料检索、掷骰与 COC 7版核心判定。

> **技术栈**：TypeScript + React/Vite + Express + Socket.IO + SQLite (sql.js) + DeepSeek API

> **最后更新**：2026-05-27
